### Authorization

<ParamField path="Authorization" type="string" required>
  Cabeçalho de autenticação Bearer no formato `Bearer <api-key>` onde `<api-key>` é a sua chave de API.
</ParamField>

<RequestExample>

```bash Criar QRCode PIX
curl --request POST \
  --url https://pay.valora.cash/v1/pixQrCode/check \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --body '{
  "txid": "83106dc5928047f395b2b5d551c9b3c3"
}'
```

</RequestExample>

<ResponseExample>
```json 200
{ 
	"status": "Paga",
	"created_at": "2025-30-10T18:56:15.538Z",
	"updated_at": "2025-30-10T18:56:15.538Z",
	"expires_at": "2025-30-10T18:56:15.538Z"
}
```


```json 401
{
	"name": "AUTH_MISSING_KEY",
	"message": "Chave de API ausente. Use o header Authorization: Bearer <API_KEY>."
}
```
</ResponseExample>
